Traffic Flow Simulation

A traffic modeling project based on real-world street observations, rebuilt into a full-stack demo using a C simulation engine, a Python API, and a browser visualizer that shows streets, cars, lights, and congestion tick by tick.

Process story
Real-world observation: street size approximations and vehicle counts from actual roads.
C systems design: structs, queues, timing logic, and probabilistic routing.
Testing: low/high represented seconds, full queues, empty queues, blocked movement, and stop-sign behavior.
Rebuild: Python backend for execution and a browser visualizer to make the simulation easier to explain.
